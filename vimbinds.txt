#Vim

:sp "name" to split window
shift ++ to comment
yw/dw to copy/delete word
/searchtext and next by “n” and shift n goes up
f  forward (f” jumps to the first “) 
.  repeat last command
space n to open new file
space a to copy everything
space s to source
space i / o to go to prevs and next file when in vim
space q to quit
space e to find files(nerd tree)

ps aux -> Activity Monitor in Terminal

F3 - Compile
F8 - Run without errors
F9 - Run with errors but no input
F10 - Run with inp



:Prog to open Library in vim (nerd tree)
